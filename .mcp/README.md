# MCP Configuration

This directory contains Model Context Protocol (MCP) server configuration for Velocity GP.

## Files

- `server.config.json` — Tool and resource definitions for the MCP server

## Integration

When MCP server is implemented (see `docs/architecture/Tech Stack Needed.md`), this configuration will define:

- **Tools**: API endpoints exposed as MCP tools
- **Resources**: Shared state and context available to AI assistants
